**FurtherHub** is a modern Python-based desktop application designed to bridge the gap between students and academic resources. It aims to eliminate the traditional "middleman" (class representative) delay by allowing students to share and access materials directly in an organized, peer-to-peer environment.

> **Note:** The application interface is fully in Arabic, tailored specifically for computer science students.

---

## âœ¨ Key Features
- **Direct Peer Sharing:** No more waiting for "Al-Mandoub"! Students can upload summaries and links directly to the platform.
- **Smart Categorization:** Resources are organized by subject, type (YouTube, PDFs, Exam Samples), and major.
- **Dynamic Rating System:** Integrated star-rating and comments system to ensure the quality and reliability of shared materials.
- **Modern UI/UX:** Built with a dark/light mode modern interface for a comfortable user experience.
- **Lightweight & Fast:** Powered by an optimized SQLite database.
